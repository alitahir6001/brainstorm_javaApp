# Brainstorm - Java web app

A full stack (CRUD) app to create ideas and their creator.

Features:
  - Login/Registration page with corresponding error messages
  - Dashboard page for logged in user, highlighting all ideas created in the DB and corresponding user
  - Only able to edit and delete the ideas of the logged in user, read-only for other users' ideas.
